What Is an Order?
-----------------
An order is an instruction to buy or sell something, like a stock.

Basic Features of an Order:
---------------------------

1. ID
   - A unique number that identifies the order.

2. Type
   - Buy  → You want to buy something.
   - Sell → You want to sell something.

3. Price
   - The price you’re willing to pay (buy) or accept (sell).

4. Quantity
   - How many units you want to buy or sell.

5. Status
   - New            → Just created
   - Filled         → Completed
   - Partially filled → Some of it is done
   - Canceled       → You took it back
